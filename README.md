# 小航 · 郑州航院校园信息查询 AI 助手

## 项目简介

"小航" 是一款基于 Streamlit + 硅基流动大模型 API 的校园信息查询 AI 助手，为郑州航院师生提供智能问答服务，覆盖新生入学、办事流程、电话黄页、应急防骗、交通出行五大场景。

## 快速开始

### 环境要求

- Python 3.8+
- 硅基流动 API Key

### 安装与运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py

# 或使用模块化版本
streamlit run src/app.py
```

### 配置

复制 `.env.example` 为 `.env`，填入你的 API Key：

```env
API_KEY=your_siliconflow_key_here
```

## 功能清单

### 核心功能
- 🎯 **身份分流**：支持新生/在校生/教师三种身份，切换不同回答风格
- 💬 **AI 问答**：基于校园 Markdown 知识库的智能问答，末尾标注来源
- 📑 **推荐问题标签页**：14 个推荐问题分 3 个标签页（新生指南/办事流程/应急防骗）
- 🗣️ **多轮对话**：对话上下文记忆，支持追问
- 📞 **电话黄页**：静态兜底页面，AI 不可用时也能查看关键号码

### 增强功能
- 📋 **历史记录**：每次问答自动保存，支持查看/清空/导出
- 🔒 **处理锁**：模型思考期间禁用按钮，防止打断
- 🌐 **断网提示**：自动检测网络状态，离线时友好提示
- 📊 **回答元信息**：显示字数、耗时、Token 消耗
- 🔄 **API 容错**：超时自动重试，429 限流递增等待

### 数据扩展
- 📂 **5 个知识库文件**：自动加载 `data/` 下所有 `.md` 文件，新增无需改代码

## 项目结构

```
xiaohang_helper/
├── src/                    # 源代码
│   ├── app.py              # Streamlit 主程序
│   ├── api.py              # API 调用（含重试/429处理）
│   ├── config.py           # 集中配置
│   ├── prompts.py          # Prompt 工程
│   └── history.py          # 历史记录持久化
├── data/                   # 知识库（5 个 md 文件）
├── docs/                   # 文档
├── screenshots/            # 项目截图
├── assets/                 # 静态资源（校徽）
├── app.py                  # 独立版入口
└── requirements.txt
```

## 技术栈

- **前端**：Streamlit
- **AI 模型**：硅基流动 GLM-5.2
- **数据存储**：Markdown 文件 + JSON 历史记录
- **核心技术**：Prompt 工程（三套身份分流 + 6 条硬规则 + 8 组别名词典）
