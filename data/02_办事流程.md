## 在校生办事

### 在校学籍证明（在读证明）
- 办理地点：综合楼一楼教务处学籍科（龙子湖校区）
- 所需材料：学生证、身份证
- 办理时间：周一至周五 8:30-11:30, 14:30-17:00
- 办结时间：当场办理
- 费用：免费
- 也可通过学校教务系统自助打印（综合楼一楼自助机）

### 校园一卡通补办
- 办理地点：综合楼一楼卡务中心（龙子湖校区）
- 所需材料：身份证或学生证
- 补办费用：20元工本费 ⚠ 以官方为准
- 办结时间：当场办理，即时生效
- 挂失方法：通过"完美校园"APP自助挂失，或到卡务中心现场挂失

### 成绩单打印
- 在校成绩单：综合楼一楼自助打印机，刷校园卡打印
- 毕业成绩单：联系教务处学籍科
- 中英文成绩单：教务处学籍科申请，3个工作日 ⚠ 以官方为准

### 转专业
- 申请时间：大一第二学期开学前两周
- 基本条件：第一学期无挂科，平均绩点达到转入专业要求
- 流程：填写转专业申请表 -> 辅导员签字 -> 学院审批 -> 教务处审核 -> 公示
- 注意事项：每个专业转入名额有限，具体以当年通知为准 ⚠ 以官方为准

### 图书馆
- 开放时间：周一至周日 7:00-22:00（寒暑假另行通知）
- 借阅规则：本科生可借10册，借期30天，可续借一次
- 图书馆位置：龙子湖校区图书馆（中心广场北侧）
- 图书馆网站：可通过学校官网进入图书馆电子资源

### 学生证补办
- 办理地点：各学院学生工作办公室
- 所需材料：一寸照片1张、身份证
- 费用：10元 ⚠ 以官方为准
- 办结时间：3-5个工作日

## 教师常用办事

### 差旅报销
- 主管部门：财务处（综合楼二楼）
- 报销流程：
  1. 出差前填写《出差审批单》，经部门负责人审批
  2. 出差结束后整理票据（交通票、住宿发票等）
  3. 填写《差旅费报销单》
  4. 部门负责人签字 -> 财务处审核 -> 报销到账
- 报销标准：按学校差旅费管理办法执行 ⚠ 以官方为准
- 办理时间：周一至周五 8:30-11:30, 14:30-17:00

### 调课申请
- 主管部门：教务处教务科
- 流程：
  1. 登录教务管理系统
  2. 提交调课申请，填写调课原因和新时间
  3. 学院教学副院长审批
  4. 教务处审核通过后生效
- 注意事项：调课需提前至少2天申请，临时调课需电话通知学生

### 教室设备报修
- 多媒体设备故障：联系教育技术中心
- 报修电话：0371-61912500（龙子湖校区） ⚠ 以官方为准
- 桌椅门窗损坏：联系后勤管理处
- 报修电话：0371-61912800（龙子湖校区） ⚠ 以官方为准

### 科研项目申报
- 主管部门：科研处（综合楼三楼）
- 申报流程：
  1. 关注科研处官网通知，了解申报时间节点
  2. 按要求准备申报材料（申报书、预算表等）
  3. 学院审核推荐
  4. 科研处汇总上报
- 常见项目类型：国家自然科学基金、河南省科技攻关项目、教育厅科研项目
- 咨询电话：0371-61912160 ⚠ 以官方为准
